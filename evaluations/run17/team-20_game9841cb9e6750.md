## Phase 1 — Rule Comprehension

### Board structure
- **3D grid** of axis size 4 → **64 cells**, von-Neumann neighbourhood (max degree 6).
- Cell index encoding: `cell = z·16 + y·4 + x`.
- 65 actions total: cells `0..63` are placements; action `64` is **pass**; two consecutive passes = **draw**.

### Turn structure
- **Alternating**, 1 piece per turn. P1 always opens. No simultaneous play.

### Action types
- **Place only.** No movement (action_types == ['place']). Action space is 65, no hybrid bloat.

### Placement constraint
- `target=empty, constraint=anywhere, first_move_anywhere=true`. You may place on any empty cell. Super-ko illegal positions are filtered.

### Capture rule
- `surround` capture (Go-style: removed enemy groups with zero liberties adjacent to the placed cell). The `threshold=1` parameter is the surround threshold from older rule schemas; not used by `_capture_surround`. **Surround capture is effectively vestigial here**: in a 3D 4³ grid every group has high liberty count (corner=3, edge=4, face=5, interior=6), so the surround condition almost never fires in normal play. In all 4 games I played to completion (1 hand + 3 greedy, ~90 placements total) zero captures fired.

### Propagation
- `influence`, **radius 2**, **strength 1.4027**, **decay 0.4724**. Each placed stone adds signed contributions to all cells within Manhattan distance ≤ 2:
  - d=0 (self): ±1.40
  - d=1 (≤6 cells): ±0.66
  - d=2 (≤18 cells): ±0.31
  - P1 contributes +, P2 contributes −. `board_values` is signed running sum, clamped to [−100, 100].

### Win condition
- `threshold` win at **42.0804**. P1's "effective" = +Σ board_values over cells P1 owns; P2's effective = −Σ board_values over cells P2 owns. First player whose effective > 42.08 (and whose margin beats opponent by FP-tolerance) wins. Both > threshold same tick → higher-margin wins, tie → draw.
- `target_dimension` (0) and `target_dimension_p2` (−1) are unused for threshold-type wins (those are connection fields).
- **max_turns=100** with majority fallback if reached.

### Degeneracy / sanity flags
- **Surround capture is vestigial** in this 3D substrate — confirmed empirically (0/4 games triggered any).
- Threshold (42.08) is reachable in ~22–26 turns of greedy play, well below max_turns 100. Not unreachable.
- No CA, no movement, no holes/sierpinski.

## Phase 3 — Joint Strategic Analysis

**Distinct viable strategies?** Two emerge — *cluster-amplification* and *halo-disruption* — but they are not symmetric in viability. Cluster-amp dominates as the leader's strategy; disruption is a defensive option that delays loss. Greedy 1-ply collapses both to "play the highest own_eff − opp_eff cell." 100% P1 wins under greedy/greedy is strong evidence one strategy dominates.

**Meaningful counter-play?** Limited. Because surround capture rarely fires (high 3D liberty count), there is no way for P2 to actually remove P1 stones — only to slow their value growth. Counter-play exists tactically (Move 20 (1,3,2) blocked P1's threshold-clincher for one turn) but never structurally.

**Short vs long term?** A weak version exists: an early disruption move trades −0.5 P2 effective for −1.5 P1 effective, but the P2 stone occupies a cell that could later be a high-amp cluster cell. Ranking these decisions is interesting for the first ~10 moves, then the race dominates.

**Emergent concepts:**
- **Halo amplification** — each new stone within radius 2 of N own stones adds ~1.4 + 0.66·(d=1 own) + 0.31·(d=2 own); clusters scale super-linearly.
- **Tempo** — P1 always reaches threshold first under symmetric play.
- **Disruption tax** — placing in opponent halo costs you cell-self-value but extracts a higher cost from opponent boost.
- No ko fights (because surround inert). No mutual annihilation. No territory in the Go sense — all cells contribute via influence regardless of ownership.
- **3D path-routing? Sub-volume control?** Neither emerges. The game is a *value race* not a *connection or territory game*.

**Does 3D enable strategies that 2D wouldn't?** No. The 3D 4³ has 64 cells; a 2D 8×8 has 64 cells; an 8×8 grid with the same r=2 propagation would have similar dynamics. 3D's only effect is increasing average degree (6 vs 4) and slightly increasing halo size (≈25 cells per stone vs ≈13). The mechanic doesn't exploit "stacking" or "layered planes" or any 3D-specific structure. The game would flatten to 2D without loss of strategic depth.

**Topology matter?** Within 3D: not really tested (only grid played). Surround capture would be more meaningful on a higher-degree topology only if degree raised liberty counts further (worse), or on a sparse topology (sierpinski/holes) where liberties are scarcer (better). This game is grid-only.

**Move-enabled?** No. Action space is place-only.

**First-mover advantage:** **Severe under shallow play** (greedy/greedy P1 wins 20/20). The training runs report final winrate 0.5 for P1 vs P2, which suggests trained PPO agents converge to a different equilibrium — but greedy 1-ply finds a clear P1 win. The seat-balance fitness gate (>0.1, hard-zero below) was set after R16 specifically to kill threshold-race winners; that this game still passed suggests the trained-agent winrate is balanced even though strategic-depth (0.48) is mediocre and greedy-first-move-advantage is total.

## Phase 4 — Novelty Adversary (mandatory)

### Adversary's case (this game is NOT novel)

**(a) Catalog comparison.** The mechanic decomposes as:
- *Place stones, signed influence accumulates over local radius* — this is a well-known abstract pattern (commercial games: *Push Fight*, *Pressure* (digital), *Splice*, "field" games; academic: any value-density placement game).
- *Race to a fixed value threshold on summed influence* — appears in scoring abstracts, including Go-style "first to 60 points wins" variants.
- *Surround-capture* — pure Go.
Putting them together: **"Othello/Reversi-with-influence-fields in 3D, threshold-race scoring."** Not a single named game, but every component is canonical.

**(b) CA literature:** N/A — no CA.

**(c) Simultaneous-game analogs:** N/A — alternating.

**(d) 3D + connection + custodian:** N/A — this is 3D + threshold + surround, not the R17 champion's recipe. Closer to "3D Othello variants" — *Spheres* (1986), *3D Othello* (Tomy), *Pulsar* (Fonseca) — though those use flip-capture not threshold-on-influence.

**(e) Re-skin?** Yes, the game is essentially *2D radial-influence value-race with redundant Go-capture decoration, lifted to 3D 4³ for cosmetic novelty*. The +5% non-grid novelty multiplier doesn't apply (grid topology). The 3D substrate doesn't justify itself: the same game in 2D 8×8 plays the same.

**(f) Expert transfer:** A Reversi expert + a Go expert combined would learn this in 5 games. Tempo, halo-amp, and disruption tax are immediately legible. There is no esoteric concept to acquire.

### Player rebuttal

- **Threshold race ≠ Othello.** Reversi flips opponent stones on placement; here capture is inert and stones never flip. The mechanic of "I score from value at cells *I* own; you score from value at cells *you* own" with no flipping is genuinely uncommon. There is no Reversi-or-Go where the winning condition is a signed-effective-sum threshold race.
- **Halo-amplification dynamics are specific.** The decay 0.47 with radius 2 produces exactly the cluster scaling 1.4 + 6×0.66 + 18×0.31 = 11.5 per fully-amplified interior cell, where decay 0.5 with r=3 (a more standard tuning) would give a different equilibrium. The game's "right number of stones to reach threshold" (~12 each side) is a tuned parameter, not a generic.
- **3D matters for halo overlap.** In 2D radius 2, two stones at d=2 contribute to ~5 shared cells; in 3D ~9. The amplification leverage is meaningfully larger, so cluster-amp dominates more strongly. (This actually makes the game *worse*, but it does differentiate from 2D analog.)
- **Disruption tax** — Move 20 in Game 1 (P2 at (1,3,2)) is a specific tactic where P2 sacrifices −0.07 own effective for −1.59 P1 effective. This trade-off doesn't exist in flip-capture games (Othello disruption either flips or fails) and doesn't exist in pure value-race games without local interaction. So this is a genuine emergent micro-pattern.

But the rebuttal is thin. The dominant gameplay is "race to 42 by clustering stones" — recognisable to anyone who has played any value-accumulation abstract.

### Novelty score: **3/10**

Components are familiar (Go-surround, value influence, threshold). The combination is unusual but not mechanically distinctive — it would be at home in a generic family of "place-and-accumulate" abstracts. The 3D substrate is decorative.

## Phase 5 — Verdict

**Team ID:** team-20
**Game ID:** 9841cb9e6750
**Rules summary:** 3D 4³ grid, alternating place-only, surround capture (vestigial), radius-2 signed influence propagation (s=1.4, d=0.47), threshold-race win at 42.08 effective.
**Topology:** 3D grid 4×4×4 (64 cells, von-Neumann neighbourhood, max degree 6).
**Turn structure:** alternating, 1 piece/turn.
**Hybrid actions:** no — place-only.

### Scores (1–10)

- **Strategic Depth: 3** — Greedy 1-ply solves it (P1 wins 20/20 greedy/greedy). Surround capture never fires. Disruption is the only counter-play and it can only delay, not reverse, the leader's threshold race. Reported strategic_depth 0.48 looks high relative to the actual race-driven dynamics; likely from move-count noise.
- **Emergent Complexity: 3** — Halo amplification and disruption-tax are the only emergent micro-patterns. No ko, no mutual annihilation, no territory, no path-routing.
- **Balance: 3** — *Severe* first-mover bias under shallow play: greedy/greedy P1=100%; my hand-played game (with conscious P2 disruption) P1 still won by +12.4 effective. Training runs converge to 50/50, suggesting trained PPO finds a more sophisticated equilibrium, but the structural bias is real and shallow agents can't overcome it.
- **Novelty (post-adversary): 3** — Components are all known. Combination is unusual but not mechanically distinctive. Not "X on a hex board" but close to "Reversi + Go-surround + value-threshold in 3D" — a recognisable family.
- **Replayability: 3** — Greedy converges on the same opening every time. Hand-play offers ~10 moves of interesting positional choice before the race becomes mechanical.
- **Overall "Would I play this again?": 3**

### Closest known-game analog
**Reversi/Othello with positional influence in 3D**, with Go-style surround capture grafted on as a (vestigial) capture mechanic. Not identical because: (i) no flip-capture, the only capture mechanic is Go-surround which never fires; (ii) winning condition is threshold-race on signed-effective rather than majority count or territory; (iii) influence radius 2 + decay 0.47 is a specific tuning that the canonical games don't share.

### Killer flaws
1. **Surround capture is vestigial.** In 3D 4³ with von-Neumann adjacency, group liberties are abundant (corner=3, edge=4, face=5, interior=6). In ~90 placements across my 4 played games, zero captures fired. The game is effectively a pure influence-race with a dead capture rule.
2. **First-mover advantage is total under shallow play.** Greedy 1-ply: P1=20, P2=0. The threshold-parity-shift fix (R17 change) didn't kill it for shallow agents.
3. **Reduces to a counting race.** After ~8 moves of opening choice, both players are on autopilot extending halos. Move sequences are interchangeable beyond rough constraints.
4. **3D adds no strategic content.** The game flattens to 2D 8×8 without losing depth. The +5% non-grid novelty multiplier doesn't apply (grid topology).

### Best quality
**Halo amplification creates legitimate positional choice in the early-mid game.** The "killer cells" — interior cells with 5–7 own stones at distance ≤ 2 — give +5 to +7 effective swings, far above the average +3 swing. Identifying and racing for these creates real (if shallow) tactics. Move 7 (cell 37 = (1,1,2)) and move 17 (cell 40 = (0,2,2)) in Game 1 illustrate this nicely.

### 3D structural contribution
**Minimal.** The 3D substrate provides 64 cells (vs 64 on 2D 8×8), max degree 6 (vs 4), and slightly larger halos (~25 cells/stone vs ~13). Strategic concepts (halo, disruption, tempo, race) all exist in 2D analogues. The game would be near-identical in 2D. This is exactly the kind of game the R17 hypothesis ("3D shift is genuinely deep, not a substrate artifact") needs to *not* be a top-10 representative — but it ranks 4th by GE. Suggests GE is over-rewarding 3D substrate alone.

### Improvement ideas
1. **Make capture meaningful.** Replace surround with custodian capture (the engine already supports it). Custodian fires on flanking and would create meaningful interaction beyond passive influence stacking, similar to the R17 champion `44f6630277b3`.
2. **Lower threshold or reduce decay.** Threshold 42.08 ≈ 12 maximally-amplified stones. Lowering to ~25 would force interaction during the build phase. Or reducing decay to 0.35 would shrink the radius-2 contribution and require denser clustering.
3. **Add seat-swap evaluation in fitness.** Current scoring rewards greedy-P1 equilibria; seat-balance gate >0.1 was passed but greedy/greedy gives 1.0 P1 winrate, indicating the metric is sampling agents that play differently than 1-ply greedy. Strengthening to "seat-swapped greedy ≤ 0.7 advantage" would have caught this.
